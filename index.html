GitHub Pages 教程

步骤 1：将文件上传到 GitHub 仓库

1. 打开刚刚创建的 GitHub 仓库：访问仓库。

2. 点击绿色的 Code 按钮，并选择 Upload files。

3. 点击 Choose your files 或者直接拖拽文件到框内。

4. 确认文件上传无误后，拉到底部，点击 Commit changes 提交。

步骤 2：启用 GitHub Pages

1. 进入仓库的 Settings 页面。

2. 在左侧菜单中，点击 Pages。

3. 在 Source 一栏，选择 main 分支，确保目录为 /（root）。

4. 保存设置，稍等片刻后，您会看到一个绿色横幅，显示网站的访问地址。

5. 点击地址，测试您的网站是否可以正常访问！

步骤 3：查看您的网站

完成设置后，打开浏览器，访问绿色横幅上的网址，即可看到您的网站内容！🎉
